# Finalizer ERP - Staj Projesi

Finalizer firmasının geliştirdiği ERP sistemlerine benzer yapıda, muhasebe ve finansal süreçleri içeren web tabanlı bir uygulama.

## Teknolojiler

- **Backend**: Node.js + Express.js
- **Frontend**: Next.js + React
- **Veri Saklama**: PostgreSql
- **Kimlik Doğrulama**: JWT

## Kurulum

1. Ana dizinde:
```bash
npm install
```

2. Backend dizininde:
```bash
cd backend
npm install
```

3. Frontend dizininde:
```bash
cd frontend
npm install
```

## Çalıştırma

Ana dizinden:
```bash
npm run dev
```

Bu komut hem backend hem frontend'i başlatır.

- Backend: http://localhost:3001
- Frontend: http://localhost:3000
